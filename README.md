# Positioning Lab — Relative, Absolute, Fixed & Sticky

A simple HTML and CSS project designed to demonstrate the different CSS positioning methods. The project includes practical examples of sticky headers, absolute positioning, fixed floating buttons, and relative positioning containers.

## Features

- Sticky header
- Relative positioning parent
- Absolute positioned badge
- Fixed floating action button
- Scrollable content area
- Practical positioning examples

## Concepts Practiced

### Position: Static
The default positioning behavior. Offset properties such as `top`, `right`, `bottom`, and `left` have no effect.

### Position: Relative
Allows an element to become a positioning reference for absolutely positioned child elements.

### Position: Absolute
Positions an element relative to its nearest positioned ancestor.

### Position: Fixed
Keeps an element fixed to the viewport even when the page is scrolled.

### Position: Sticky
Allows an element to scroll normally until it reaches a specified position, then remain visible while scrolling.
